screenCoord_webSample

A simple tool for estimating length on HTML in CSS pixels.
This is web sample version ofscreenCoord.

Length on HTML view is estimated using a distance between two clicked points.
Clicks on "available area" are detected as above clicked points.

Screenshot and details

Converting length value (CSS pixels into mm)

Activation of converter
1. Unit converter must be activated using the current clicked points log.
Activated converter can convert length value in the current log into mm.

2. A new clicked log to convert can be created after clearing the current log.
